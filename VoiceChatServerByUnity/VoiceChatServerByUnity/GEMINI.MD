# Unity로 음성채팅 프로그램 만들기

## 구성 형태
- 사용자는 중앙 서버에 접속한다.
- 중앙 서버 접속 시 닉네임을 설정하여 접속할 수 있다.
- 닉네임은 중복될 수 없다.

- 중앙 서버에서는 아래 4가지 행동을 수행할 수 있다.
  - 접속자 닉네임 확인
  - 채팅방 생성
  - 채팅방 탐색
  - 채팅방 접속
  - 중앙서버 나가기

### 채팅방 생성
- 채팅방 생성 시 제목, 최대 접속 인원수를 설정할 수 있다.
- 채팅방을 생성하면 자신이 방장이 되어 채팅을 시작할 수 있다.
  - 방장이 방을 나가면 그 방은 사라진다.

### 채팅방 탐색
- N초에 1번씩 중앙 서버로부터 생성된 채팅방의 이름과 생성한 사람의 닉네임을 받아온다.

### 채팅방 접속
- Unity에서 특정 액션을 취할 시, 해당 채팅방에 접속한다.
- 접속한 채팅방에 있는 사람들에게 음성을 전송한다.

### 중앙서버 나가기
- Unity에서 특정 액션을 취할 시, 서버에서 벗어난다. 이 때, 서버에 등록된 닉네임은 삭제되어야 한다.
